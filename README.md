# 🌊 FloodSense – AI Flood Prediction System

FloodSense is an IoT + Machine Learning based system for real-time flood prediction using environmental sensor data.

## 🚀 Features
- Real-time sensor data collection (Arduino)
- ML-based flood prediction (Random Forest, LSTM, etc.)
- SMS alert system (Twilio)
- Web dashboard (Flask)

## 📊 Input Features
- Rainfall
- Soil Moisture
- Water Level
- Temperature
- Humidity
- Pressure

## 🧠 Models Used
- Logistic Regression
- Random Forest
- XGBoost
- LSTM (Deep Learning)
