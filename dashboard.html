🔌   SENSOR WIRING — NODEMCU ESP8266
PIN → SENSOR → CONNECTION
🌧 A0 → Rainfall Sensor (AO) → Analog in
📡 D7 → Rainfall Sensor (DO) → Digital trigger
🌊 D5 → HC-SR04 (TRIG) → Water level trigger
🌊 D6 → HC-SR04 (ECHO) → Water level echo
🌱 A1 → Soil Moisture (AOUT) → Analog in
🌡 D4 → DHT22 (DATA) → Temp/Humidity
🕑 D1 → BMP280 (SCL) → I2C Clock
📊 D2 → BMP280 (SDA) → I2C Data
⚡ 3.3V → Rainfall, DHT22, BMP280, Soil Moisture VCC
⚡ 5V → HC-SR04 VCC
⏚ GND → All sensor GND
📦   COMPONENTS LIST
1× NodeMCU ESP8266 (or ESP32)
1× Rainfall Sensor Module (YL-83 / FC-37)
1× HC-SR04 Ultrasonic Sensor (Water Level)
1× Capacitive Soil Moisture Sensor v1.2
1× DHT22 Temperature & Humidity Sensor
1× BMP280 Barometric Pressure Sensor
1× 10kΩ resistor (DHT22 pull-up)
1× Breadboard + Jumper Wires
1× USB Power / LiPo Battery
⚠️ Use voltage level shifter if mixing 5V and 3.3V logic.
Capacitive soil moisture sensor preferred over resistive (longer life).
